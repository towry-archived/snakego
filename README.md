greedy snake game in Go
-----------------------

greedy snake game in Go

### Usage

```bash
go get github.com/towry/snakego
make
make run
```

#### Keys

Press <kbd>Space</kbd> to stop, press <kbd>Space</kbd> again to resume.

Use arrow key to control the direction of snake.
